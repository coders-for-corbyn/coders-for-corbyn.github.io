Getting Started

Steps for Joining

  1. Read our code of conduct and manifesto
  2. Introduce yourself on the Facebook group: https://www.facebook.com/groups/codersforcorbyn
  3. Discuss new/ongoing projects on Slack: https://codersforcorbyn.slack.com
  4. Join the Trello: https://trello.com/b/FTiw5SWJ/coders-4-corbyn
  5. Get coding!

Contributors Guide

  1. Comment code as you go, this will make life easier for somebody that steps into a project you have been working on.
  2. If you are editing somebody else’s code, follow the same style/syntax and try to avoid cleaning somebody else’s code. Edit lines relevant to your task for a simpler git merge. Refactoring is a separate job.
  3. Communicate with others if you are unsure. Join the Slack channel, post on Facebook, whatever makes you feel most comfortable. There is no question too silly.
